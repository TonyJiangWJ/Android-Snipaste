# 截图贴图工具

- 基于AutoJS实现的Android端截图贴图工具，类似PC端的Snipaste工具。可以截图并截取一部分，使用悬浮窗展示截取的小图
- 直接安装的版本见[Releases](https://github.com/TonyJiangWJ/Android-Snipaste/releases)
